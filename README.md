# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/14_tsrwFwJOAwp97GuPjnEM0LIc91GQG8

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set `GROQ_API_KEY` in `.env` or your Vercel project environment
3. Run the app:
   `npm run dev`
   
   To test the API routes locally, run `vercel dev` instead.
